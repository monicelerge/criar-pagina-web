Como funciona o HTML

O HTML (Hypertext Markup Language) funciona como um sistema de marcação que estrutura o conteúdo de uma pagina web.

Ele utiliza tags,que são palavras chaves que indicam ao navegador como o conteúdo deve ser exibido, como um título,um parágrafo,uma lista, etc.

Para ficar mais fácil para entender quando imaginamos que o HTML representa o esqueleto dos sites, enquanto o JavaScript seria os músculos que dão movimento e o CSS as característica físicas.

O que são Clients?

Componente que interage com o código HTML de uma página web e o exibe no navegador.

É o software que recebe o HTML do servidor, interpreta-o e o apresenta ao utilizador.

Por exemplo, um navegador web (como o Chrome,Firefox, Safari ou Edge)

O que são Servers?

Servidor web ou servidor HTML é um computador ou sistema que armazena e entrega arquivos HTML, bem como outros recursos de imagens, CSS e JavaScript,para navegadores web quando solicitados por usuários.

Eles são a base para a experiência online,garantindo que o conteúdo seja entregue de forma eficiente e confiável.

O que são tags?

Também chamadas de elementos são a base para construir a estrutura e o conteúdo de uma página da web.

São como comandos que informam ao navegador como exibir o conteúdo, marcando o ínicio e o fim de diferentes elementos.

  • Tags de formatação:

    • (b),(i),(u),(pr e), etc.

  • Tags de estrutura:

    • (html),(head),(body),(header),(na),(main),(aside),(footer), etc.

  • Tags de conteúdo:

    • (p),(h1),(h6),(div),(span),(ul), (ol),(li),(img),(a), etc.

Entendendo a diferença entre listas ordenadas e não ordenadas

A principal diferença entre a lista ordenada e não ordenada está na maneira com que os itens de cada lista são apresentados.

O ol é uasado na lista ordenada essa lista enumera os elementos.

O ul é usado na lista desordenada ou lista não ordenada,ela coloca marcadores antes de cada elemento.
